# gRPC Services

ARQENOR uses Protocol Buffers 3 (proto3) over gRPC for typed, streaming IPC between the Rust host-analysis engine and any consumer (Go orchestrator, TUI, CLI, external tools).

Proto files live in `proto/`. Rust stubs are auto-generated by `arqenor-grpc/build.rs`. Go stubs are generated by `scripts/gen-proto.ps1`.

---

## Proto files

```
proto/
├── common.proto           # Shared enums and messages
├── host_analyzer.proto    # Host analysis RPCs
└── network_scanner.proto  # Network scan RPCs (Phase 3)
```

message ProcessInfo {

  string          exe_path       = 4;

#### ProcessEvent

```protobuf
enum ProcessEventKind {
  CREATED    = 0;
  TERMINATED = 1;
  MODIFIED   = 2;
}

message ProcessEvent {
  ProcessEventKind kind      = 1;
  ProcessInfo      process   = 2;
  int64            timestamp = 3;
}
```

#### FileEvent

```protobuf
enum FileEventKind {
  FILE_CREATED  = 0;
  FILE_MODIFIED = 1;
  FILE_DELETED  = 2;
  FILE_RENAMED  = 3;
}

message FileEvent {
  FileEventKind kind      = 1;
  string        path      = 2;
  string        hash      = 3;   // SHA-256 hex, empty for deletions
  uint64        size      = 4;
}
```

#### PersistenceEntry

```protobuf
message PersistenceEntry {
  string kind     = 1;   // matches PersistenceKind enum names
  string name     = 2;
  string command  = 3;
  string location = 4;
  bool   is_new   = 5;   // true = not in baseline
}
```

#### ScanRequest

```protobuf
message ScanRequest {
  string          root_path  = 1;
  bool            recursive  = 2;
  repeated string extensions = 3;   // filter by extension, empty = all
  uint64          max_size   = 4;   // bytes, 0 = no limit
}
```

#### HealthResponse

```protobuf
message HealthResponse {
  string status   = 1;   // "ok" | "degraded"
  string platform = 2;   // "windows" | "linux" | "macos"
  string version  = 3;   // semver
}
```

### Service

```protobuf
service HostAnalyzer {
  rpc GetProcessSnapshot (google.protobuf.Empty) returns (ProcessSnapshot);
  rpc WatchProcesses     (google.protobuf.Empty) returns (stream ProcessEvent);
  rpc ScanFilesystem     (ScanRequest)           returns (stream FileEvent);
  rpc WatchFilesystem    (ScanRequest)           returns (stream FileEvent);
  rpc GetPersistence     (google.protobuf.Empty) returns (PersistenceList);
  rpc Health             (google.protobuf.Empty) returns (HealthResponse);
}
```

## Code generation

### Rust (automatic)

`arqenor-grpc/build.rs` runs on every `cargo build -p arqenor-grpc`:

```rust
fn main() -> Result<(), Box<dyn std::error::Error>> {
    tonic_build::configure()
        .out_dir("src/generated")
        .compile(&["../../proto/host_analyzer.proto", ...], &["../../proto"])?;
    Ok(())
}
```

Generated files in `src/generated/` — **do not edit manually**.

### Go

Run `scripts/gen-proto.ps1` (PowerShell) or equivalent:

```powershell
protoc \
  --proto_path=proto \
  --go_out=go/internal/grpc/generated \
  --go_opt=paths=source_relative \
  --go-grpc_out=go/internal/grpc/generated \
  --go-grpc_opt=paths=source_relative \
  proto/common.proto proto/host_analyzer.proto proto/network_scanner.proto
```

Prerequisites for Go codegen:
```bash
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
```

## Connecting a new client

Any language with a gRPC library can connect. Example in Python (grpcio):

```python
import grpc
from generated import host_analyzer_pb2_grpc, host_analyzer_pb2
from google.protobuf import empty_pb2

channel = grpc.insecure_channel('127.0.0.1:50051')
stub = host_analyzer_pb2_grpc.HostAnalyzerStub(channel)

snapshot = stub.GetProcessSnapshot(empty_pb2.Empty())
for proc in snapshot.processes:
    print(proc.pid, proc.name, proc.sha256)
```
